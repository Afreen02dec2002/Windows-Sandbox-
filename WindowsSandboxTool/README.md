# Windows Sandbox Tool
## Trustworthy Computing Assignment

### Project Overview
This tool provides a secure environment for executing untrusted code using Windows Sandbox with configurable security policies.

### Features Implemented
- ✓ **Step 1**: Test program (Hello.exe) with console I/O
- ✓ **Step 2**: Windows Sandbox integration with configuration files
- ✓ **Step 3**: Security policy restrictions (Network, GPU, Audio, etc.)
- ✓ **Step 4**: Parameterized command-line interface
- ✓ **Step 5**: Graphical user interface with Forms
- ✓ **Step 6**: Complete documentation and user manual
